In my past semester of my third year I did the modules Software Engineering I, Databases Principles and Applications, and Multimedia and Human-Computer-Interaction (HCI). We did a few projects this semester, a web application (a.k.a a website), it is VERY arbitrary and VERY MUCH a prototype, however it allowed me to wet my feet in web dev, a group project where we designed a software system for a fictional company (which will be implemented in the coming semester, with the same group members) and making a little database in mySQL from some information provided in an assignment brief.
